  • Loads IFC into a SceneModel and/or DataModel.

    • Expects SceneModel.built and SceneModel.destroyed to be false
    • Does not call SceneModel.build - we call that ourselves, when we have finished building the SceneModel

    See @xeokit/webifc for usage.

    Returns

    Resolves when IFC has been loaded into the SceneModel and/or DataModel.

    Throws

    SDKError

    • If the SceneModel has already been destroyed.
    • If the SceneModel has already been built.
    • If the DataModel has already been destroyed.
    • If the DataModel has already been built.
