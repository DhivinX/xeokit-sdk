xeokit Geometry Builder Functions

Functions to generate various 3D geometric primitives

Installation

Usage

 
import {buildBoxGeometry} from "@xeokit/procgen";

const geometryArrays = buildBoxGeometry({
    center: [0,0,0],
    xSize: 1,  // Half-size on each axis
    ySize: 1,
    zSize: 1
});
